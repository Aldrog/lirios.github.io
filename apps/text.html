Liri Text

About

Liri Text is a cross-platform, Material Design text editor.

We aim to create a simple and beautiful text editor with all the features you would expect from a modern editor. By designing Liri Text according to Google's Material Design we where able to create a clean and beautiful interface.

Features

Simple

Editor that makes it's job

We try to keep everything simple yet functional so you can focus on what on stuff that really matters.

Yet Powerful

You'll find every feature you would expect to see

Liri Text has all the features essential for text/code editor, including syntax highlighting for most of widespread languages. We archieve this by implementing GtkSourceView standard for syntax specifications.

Contribute

Liri Text is community-driven open-source project. We rely on contributions to keep the project sustainable. Don't think you can only contribute if you know how to code! There are plenty of ways to help out:

Design

Everything starts with an idea. If you're a designer, your contribution in form of mockups, sketches or other graphics are from great value for the whole project. Checkout the GitHub issue list and get in touch by joining our Community on Google+.

Code

Writing code is without any doubt a fantastic way to contribute to the development of the project. Liri Text is written using Qt (C++) for app logics and QtQuick (QML, JavaScript) for UI. We use QtQuick Controls 2 and Fluid to create our Material Design based user interface.

git clone https://github.com/lirios/text Fork us on GitHub
Translations

We want to build an editor for everyone. In order to reach this goal, we need your help translating Liri Text into your language!
